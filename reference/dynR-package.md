# dynR: Dynamic Connectivity Analysis for Neurophysiological Timeseries

An R port of the Python dynfc library for computing dynamic functional
connectivity representations from multivariate neurophysiological
timeseries, including BOLD fMRI, EEG, LFP, and related signals.
